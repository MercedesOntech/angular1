Herramientas y tecnologías utilizadas

  • Angular 21 (Standalone Components)
  • TypeScript 5+
  • Bootstrap 5.3
  • XAMPP / Docker (para despliegue local y contenedores)
  • Visual Studio Code + extensiones Angular/TypeScript
  • Base de datos local: WebStorage / localStorage (simulación de eventos y usuarios)

Directivas utilizadas

Directiva Tipo Componente usado
*ngIf Control de flujo Header, Login, Eventos (mensaje "No hay eventos")
*ngFor Control de flujo Home (imágenes), Eventos (lista de eventos)
*ngSwitch / *ngSwitchCase / *ngSwitchDefault Control de flujo CrearEvento (mensajes de validación de campos)
[ngClass] Estilo dinámico Login, CrearEvento
[ngStyle] Estilo dinámico Eventos (precio en rojo si >50€, verde si <=50€)
Pipe personalizada (mayusculas) Transformación de datos Eventos (titulo)
Pipes integradas (DatePipe, CurrencyPipe) Transformación de datos Eventos (fecha y precio)
@Input (opcional) Comunicación padre → hijo CrearEvento (se puede usar para editar datos desde un componente padre)
@Output (opcional) Comunicación hijo → padre CrearEvento (podría usarse para notificar creación de evento)
